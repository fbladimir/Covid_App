Covid 19 - Data Helper

Here is the dashboard where you can select what you want to do.

DASHBOARD

Patients

Add, edit, or delete any patient from the database (ID Number, Full Name, and Symptoms required to add any patients).

Appointments

Create, edit, or delete appointments for the patients if they want to get tested for Covid-19.

Results

Check the results of the tests for the patients.

Symptoms

See all the symptoms for Covid-19 to keep everyone informed, or add new ones.

Notes

Create, edit, or delete notes related to the Covid-19 pandemic to keep everyone informed.

Settings

Change the settings of the Web-App.
